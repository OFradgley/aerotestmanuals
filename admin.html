Quick Instructions

To Add New Product Manuals:

  1. Upload Files: Place PDF manuals and product images in the respective product folders
  2. Update Links: Edit the index.html file in each product folder to link to your uploaded files
  3. Add New Products: Copy existing folder structure and modify the main index.html to add new product cards

File Management:

  • Images: Place product photos in the product folders (recommended: .jpg format)
  • PDFs: Upload manual PDFs to the respective product folders
  • Updates: Use the "Manage Files" buttons above to access product folders directly
