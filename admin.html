Instructions

📁 Adding a New Product:

  1. Create Folder: Create a new folder (e.g., "AT1520") in the main directory
  2. Copy Template: Copy the index.html file from an existing project folder (e.g., "AT1175") into your new folder
  3. Edit Product Page: Open the new index.html and update:
    • Page title: Change "AT1175" to your new product code (line 191)
    • Product title: Change the h1 heading "ATL1175-001 C-130J QEC WVR with Accessible Tail-Pipe “Aero-bag”" to your new product title (line 218)
    • Manual Link: Change the manual link "ATL1175-001 WVR Aero Bag C-130J QEC Installation Instructions Issue 11.pdf" to your new manual PDF title (line 261)
    • Update the manual issue number (line 261)
  4. Add to Admin Panel: Copy/paste a product-item block and rename in the admin.html file
  5. Upload Files: Add your product image ("image.jpg") and PDF manual to the new folder
